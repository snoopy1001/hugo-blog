## 版本信息

pandas 中的 show_versions 函数可以显示 pandas 及其相关的依赖库的版本号。

查看结果如下：

| 名称       | 版本   | 说明     |
| ---------- | ------ | -------- |
| python     | 3.7.6  | 编程语言 |
| IPython    | 7.12.0 | 编辑交互 |
| pandas     | 1.0.1  | 数据分析 |
| numpy      | 1.18.1 | 数组运算 |
| matplotlib | 3.1.3  | 图像显示 |
| sklearn    | 0.22.1 | 机器学习 |
| scipy      | 1.4.1  | 科学计算 |
